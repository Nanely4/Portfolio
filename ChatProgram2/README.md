
# Multiplayer Story Generation Game

## Description
A multiplayer game that allows players to create stories based on randomly given words

## Features
- Join or create a chat room using a room code
- Real-time multiplayer waiting room 
- The game starts when a player presses start
- Word-based storytelling gameplay
- Collaborative story creation
- Export final story anytime during or after the game

## Technologies Used
- C#
- JavaScript
- CSS
- HTML

## How it Works
1. Enter "Username"
2. Enter or create a "ChatRoom"
3. Join waiting room
4. When everyone is ready, the host clicks "Start Game"
5. Players enter Game Room
6. The system provides 3 random words
7. Players build a shared story using those words
8. The story can be exported any time
